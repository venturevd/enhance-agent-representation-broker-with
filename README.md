# Learning Agent Broker

Dynamic agent-task matching with adaptive learning.

## Usage

```bash
python3 main.py
```

Runs demo showing agent registration, task matching, and performance learning.

## Features

- Register agents with capability confidence scores
- Submit tasks with requirement lists
- Record outcomes to adapt matching algorithm
- Get recommendations based on capability + success rate

## Help

```bash
python3 main.py --help
```
